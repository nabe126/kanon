## 3. 📝 意思決定ログ (Decision Log)

* **Docker CLI パッケージから直接 API 呼び出しへの移行**:
  * Ubuntu 26.04 環境下で Docker ビルド時に AppArmor が干渉し、`apt-get install docker.io` や静的バイナリの実行がブロックされる現象に直面。
  * `runc` の AppArmor 関連の不整合を回避し、かつコンテナイメージサイズを数百MB削減するため、ソケット通信経由で Docker Engine API を直接叩く方式を選択。これにより運用の堅牢性と軽量化を同時に達成。
* **本番用 docker-compose への monitor サービス追加**:
  * 実機で `make up` で本格稼働させる際、自動ロールバック監視がバックグラウンドで常駐稼働するよう、[ai-agent/docker-compose.yml](file:///Users/nabe/src/github.com/nabe126/kanon/ai-agent/docker-compose.yml) に `monitor` コンテナ定義を追加し、`restart: always` を設定。
