## 5. ⚠️ リスクと制約事項 (Risks)

* **機密情報の管理**:
  * `secrets/.env` 内の Discord トークンや API キーは絶対に漏洩させないこと（チャットやログに書き出さないよう厳守）。

  * 自動ロールバックによる Docker 再起動の仕組みは、予期しないデッドロックやコンテナクラッシュを引き起こすリスクがあるため、完全に検証されるまで、本番での自動起動（デーモン化）は行わず、手動起動による Experimental 監視のみに留めること。
